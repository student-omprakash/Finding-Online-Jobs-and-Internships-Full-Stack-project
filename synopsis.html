📋 Abstract

CareerNest is a full-stack web platform designed to bridge the gap between students seeking employment opportunities and recruiters looking for fresh talent. The application provides a centralised ecosystem where students can discover jobs and internships, build and manage their professional profile, and track their applications — all in one place.

Recruiters benefit from a dedicated dashboard to post openings, manage listings, and review applicants. The system is backed by a secure REST API built on Node.js/Express with MongoDB as the database, and a React-based SPA for the frontend. The project follows modern DevOps practices including Docker containerisation, GitHub Actions CI/CD, and cloud deployment on Render.

🎯 Objectives

For Students

  • Discover relevant jobs and internships
  • Build a detailed professional profile
  • Apply with a single click
  • Track application status in real time
  • Manage resume, skills, education & experience

For Recruiters

  • Post and manage job/internship listings
  • View and filter applicants
  • Update application statuses
  • Maintain a branded company presence

✨ Key Features

Authentication & Security

  • JWT-based stateless authentication
  • OTP email verification via Nodemailer
  • Password hashing with Bcrypt.js
  • Forgot / Reset Password flow
  • Role-based access (Student / Recruiter / Admin)

Job & Internship Module

  • Filter by type: Internship, Full-time, Part-time, Contract
  • Search by skills, location, experience level
  • Real-time application status tracking
  • Duplicate application prevention

Student Profile

  • Bio, contact & social links
  • Education & Work Experience timeline
  • Skills with proficiency levels
  • Certifications with URLs
  • Resume upload & management

Recruiter Dashboard

  • Create / Edit / Delete job listings
  • Toggle job open/closed status
  • View all applicants per job
  • Change application status (shortlist, interview, accept, reject)

🚀 Deployment & DevOps

Docker Setup

  • 3-service Docker Compose: MongoDB, Backend, Frontend
  • Multi-stage Dockerfile for optimised images
  • Nginx serves the React build in production
  • Named volumes for data persistence
  • Health checks on all services

Cloud Deployment

  • GitHub Actions CI/CD pipeline
  • Frontend hosted on Render (Static Site)
  • Backend hosted on Render (Web Service)
  • MongoDB Atlas for cloud database
  • Environment-based config via .env

🔒 Security Measures

API Security

  • Helmet.js HTTP headers
  • CORS whitelist
  • express-validator input validation
  • Morgan request logging

Auth Security

  • JWT tokens (httpOnly)
  • Bcrypt password hashing (salt 10)
  • Time-limited OTP for reset
  • Password field select:false

File Security

  • Multer file type validation
  • Static folder isolation (/uploads)
  • Environment secrets via .env

📝 Conclusion

CareerNest successfully demonstrates a production-grade full-stack web application addressing the real-world problem of student career discovery. It employs the MERN stack with industry-standard practices including JWT authentication, role-based access control, containerised deployment with Docker, and automated CI/CD pipelines via GitHub Actions.

The project covers every layer of modern software development — from database design and REST API architecture to responsive UI/UX, file management, email services, and cloud deployment — making it a comprehensive showcase of full-stack engineering skills.
